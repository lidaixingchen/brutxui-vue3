# Vue 3 + TypeScript + Vite

此模板可帮助你快速开始使用 Vue 3 和 TypeScript 在 Vite 中进行开发。该模板使用 Vue 3 `<script setup>` SFC，请查看 [script setup 文档](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup)了解更多。

关于推荐的项目设置和 IDE 支持，请参阅 [Vue Docs TypeScript 指南](https://vuejs.org/guide/typescript/overview.html#project-setup)。
